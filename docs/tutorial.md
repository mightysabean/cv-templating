# Tutorial

Clone the project.



To check the examples, in the directory of the program you must do, for HTML:
```sh
python mcv.py -i examples/en/html/awsome/cvhtml.jinja -t html
```
And for LaTeX:
```sh
python mcv.py -i examples/en/latex/europasscv/cvlatex.jinja -t latex
```
